# zeroday-recon

All-in-one Bug Bounty & VAPT Recon Automation Framework

## Supported OS
- Kali Linux (tested)
- Debian-based systems (partial)

## Features
- Auto tool install & check
- Subdomain enumeration
- Live host detection (httpx)
- Port scanning (naabu)
- Screenshotting (gowitness)
- Vulnerability scanning (nuclei)

## Tools Used
subfinder, assetfinder, amass, httpx, naabu, gowitness, nuclei

## Installation
1. git clone https://github.com/vigneshcybersecurity/zeroday-recon.git
2. cd zeroday-recon
3. chmod +x zeroday-recon.sh

## Usage
./zeroday-recon.sh example.com

## Output
Results are saved in:
recon-example.com-YYYY-MM-DD/
